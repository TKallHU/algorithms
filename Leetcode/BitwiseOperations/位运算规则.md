|符号|描述|运算规则|  
|---|----|----|
|&|与|两个位都为 1 时，结果才为 1 |
| &#124; |或|两个位都为 0 时，结果才为 0。换言之，只要有 1 ，结果就为 1|
|^|异或|两个位相同为 0 ， 相异为 1
|~/-| 取反|0 变 1， 1 变 0
|<<|左移|各二进位全部左移若干位， 高位丢弃，低位补 0 
|>>|右移|各二进位全部右移若干位

#### 异或操作的一些特点
```
x ^ 0 = x

x ^ 1s(全1) = ~x

1s = ~0

x ^ (~x) = 1s

x ^ x = 0

a ^ b = c       --->   a ^ c = b, b ^ c = a

a ^ b ^ c = a ^ (b ^ c) = (a ^ b) ^ c
```

#### 常用的位运算操作
```
x & 1 == 1 or 0   # 判断奇偶性（x % 2 == 1 or 0)

x & (x - 1)  # 清零最低位的1

x & -x   # 得到最低位的1
```
